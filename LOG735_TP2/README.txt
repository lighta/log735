Afin de géré nos dependances nous utilisons Maven.
Nous supposons que vous savez configurer votre environnement et que vous etes familiez avec Maven pour l'utilisation de ce lab.

Compilation serveur :
 - mvn compile

Compilation client :
 - mvn compile

Compilation archive:
 - Accessoirement vous pouvez recompiler une archive de remise en effectuant la commande : 'ant -f ant-remise-lab.xml'
