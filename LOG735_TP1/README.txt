Afin de gere nos dependances nous utilisons Maven.
Nous supposons que vous savez configurer votre environnement et que vous etes familiez avec Maven pour l'utilisation de ce lab.

Compilation serveur :
 - mvn package -Pserver

Compilation client :
 - mvn package -Pclient

Compilation archive:
 - Accessoirement vous pouvez recompiler une archive de remise en effectuant la commande : 
 -- 'ant -f ant-remise-lab.xml'
