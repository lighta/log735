Readme

Pour utilisez le systeme veuillez lancer les elements suivants :
-Banque : Banque ou banque_tag1.jar
-Succursale : SuccursaleServer ou succursale_tag1.jar

Ceci lancera des serveurs sur chacun des hotes les hebergeants.
Pour interragir avec les serveurs veuillez lancer un client, (console), (putty ou telnet).

Les succursales ecoute sur le port 9200.
Les Banques ecoutes sur les 
--port 9100 : pour les consoles
--port 9300 : pour les succursales

Une fois les connections faite entrez #HELLO pour vous identifiez.
Sur la succursale entrez !CON#hostname:port pour vous connecter a la banque.
L'ensemble des commandes pour les differents element est disponible en Annexes du rapport.
